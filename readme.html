Steps to Configure the Duo Splunk SOAR app's asset

Follow these steps to configure the Duo Splunk SOAR app's asset:

  • Role required: Owner
  • Note that only administrators with the Owner role can create or modify an Admin API application in the Duo Admin Panel.
    • Sign up for a Duo account.
    • Log in to the Duo Admin Panel and navigate to Applications.
    • Click Protect an Application and locate the entry for Admin API in the applications list. Click Protect to the far-right to configure the application and get your integration key, secret key, and API hostname.
      • Determine what permissions you want to grant to this Admin API application.
      • Give Grant read resource and Grant write resource permissions
  • Now, Log in to your Splunk SOAR instance.
    • Navigate to the Home dropdown and select Apps.
    • Search the Duo App from the search box.
    • Click on the CONFIGURE NEW ASSET button.
    • Navigate to the Asset Info tab and enter the Asset name and Asset description.
    • Navigate to the Asset Settings.
    • Paste the generated API hostname, integration key, secret key from Duo UI to its respective configuration parameter.
    • Save the asset.
    • Now, test the connectivity of the Splunk SOAR server to the Duo instance by clicking the TEST CONNECTIVITY button.
